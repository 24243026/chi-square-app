What is Chi-Square?

📖

Definition

The Chi-Square (χ²) test is a statistical hypothesis test used to examine how well observed data matches an expected distribution (Goodness of Fit) or whether two categorical variables are independent (Test of Independence).

🔢

Formula

χ² = Σ (O − E)² / E

O = Observed frequency  ·  E = Expected frequency  ·  Σ = Sum across all categories

📊

Types

  • Goodness of Fit — Does the observed data follow the expected distribution? (1 row of data, df = n−1)
  • Test of Independence — Are two categorical variables associated? (contingency table, df = (r−1)(c−1))
✅

When to Use

  • Data is in the form of counts / frequencies
  • Categories are mutually exclusive
  • Expected frequency in each cell ≥ 5
  • Observations are independent
⚖️

Decision Rule

If χ²calc > χ²critical → Reject H₀

If χ²calc ≤ χ²critical → Fail to Reject H₀

Equivalently: if p-value < α → Reject H₀

📐

Degrees of Freedom

  • Goodness of Fit: df = n − 1 (n = number of categories)
  • Independence: df = (r − 1) × (c − 1) (r = rows, c = columns)
